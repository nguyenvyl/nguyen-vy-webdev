About This Site

This website was created as my final project for Northeastern University's Web Development class. It's a work in progress, so please excuse the clutter!

All trail information is sourced from the Trails API. Note that this database is incomplete; not all trails will have detailed information and coordinates. I'll look into incorporating other publicly available trail databases in the future!

Weather information is courtesy of DarkSky.

Embedded maps courtesy of Google. I hope to add a dynamic map search in the future as well!

And finally, you can view my personal website here!

Happy hiking!
